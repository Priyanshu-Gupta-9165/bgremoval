## âš™ï¸ Installation & Usage

1. **Clone the repository**
   ```bash
   git clone https://github.com/Priyanshu-Gupta-9165/bgremoval.git
   cd bgremoval
   ```

2. **Run locally**
   Simply open `index.html` in your favorite web browser.
   
   *Tip: For the best experience, use a local server like Live Server in VS Code.*

3. **API Setup**
   The project currently uses a demo API key. To use your own:
   - Get a free key from [remove.bg](https://www.remove.bg/api).
   - Open `script.js`.
   - Replace the `API_KEY` variable with your new key.

   ```javascript
   const API_KEY = 'YOUR_API_KEY_HERE';
   ```
